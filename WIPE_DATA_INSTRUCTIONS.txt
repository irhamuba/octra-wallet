HOW TO WIPE ALL EXTENSION DATA (TOTAL RESET)
==============================================

Use the code below to completely wipe all sensitive data (Private Key, Seed Phrase, History) from the browser memory.

INSTRUCTIONS:
1. Open the UBA Wallet Extension in your browser (click the icon).
2. Right-click anywhere on the wallet interface -> select "Inspect".
3. The 'Developer Tools' window will appear. Click on the "Console" tab.
4. Copy the entire code below.
5. Paste it into the Console.
6. Press ENTER.
7. The Wallet will be reset clean as if newly installed.

--- COPY THE CODE BELOW ---

(async function wipeAllData() {
    console.clear();
    console.log("%cüî• STARTING DATA WIPE... üî•", "color: red; font-size: 20px; font-weight: bold;");

    try {
        // 1. Clear Chrome Storage (Local & Sync) - This is the main Wallet storage
        if (typeof chrome !== 'undefined' && chrome.storage) {
            await new Promise(resolve => chrome.storage.local.clear(() => {
                console.log("‚úÖ Chrome Local Storage: CLEANED");
                resolve();
            }));
            await new Promise(resolve => chrome.storage.sync.clear(() => {
                console.log("‚úÖ Chrome Sync Storage: CLEANED");
                resolve();
            }));
        } else {
            console.log("‚ö†Ô∏è Chrome API not detected (are you in a browser environment?)");
        }

        // 2. Clear LocalStorage (Standard web storage)
        localStorage.clear();
        console.log("‚úÖ LocalStorage: CLEANED");

        // 3. Clear SessionStorage (Temporary session)
        sessionStorage.clear();
        console.log("‚úÖ SessionStorage: CLEANED");

        // 4. Clear IndexedDB (Usually for large transaction history)
        if (window.indexedDB && window.indexedDB.databases) {
            const dbs = await window.indexedDB.databases();
            for (const db of dbs) {
                if (db.name) {
                    window.indexedDB.deleteDatabase(db.name);
                    console.log(`‚úÖ IndexedDB [${db.name}]: DELETED`);
                }
            }
        }
        
        console.log("%c‚ú® SUCCESS! ALL DATA HAS BEEN WIPED. ‚ú®", "color: green; font-size: 16px; font-weight: bold;");
        console.log("Please close and reopen the extension, or reload.");
        
        // Optional: Auto reload page
        setTimeout(() => window.location.reload(), 2000);

    } catch (error) {
        console.error("‚ùå An error occurred while wiping data:", error);
    }
})();
