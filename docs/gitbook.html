Overview

Octra Wallet is a browser extension wallet designed specifically for the Octra blockchain ecosystem. It provides secure, non-custodial storage and transaction capabilities directly from your browser.

Non-Custodial: You control your private keys. We never have access to your funds.

Client-Side Only: All operations happen locally on your device. No servers, no tracking.

Key Features

  • Secure Storage: AES-256-GCM encryption for wallet data
  • Multi-Wallet Support: Create and manage multiple wallets
  • Send & Receive OCT: Fast and secure transactions
  • Token Support: OCS01 token standard compatibility
  • Privacy Features: FHE encrypted balance (experimental)
  • Transaction History: View your past transactions
  • QR Code Support: Easy address sharing
  • Auto-Lock: Wallet locks after 5 minutes of inactivity

Getting Started

Installation

Currently available for:

  • Chrome/Chromium browsers (coming to Chrome Web Store)
  • Firefox (coming to Firefox Add-ons)

Creating Your First Wallet

  1. Install the extension from your browser's extension store
  2. Click "Create New Wallet" when you first open the extension
  3. Set a strong password (minimum 8 characters)
  4. Save your 12-word recovery phrase - Write it on paper and store securely
  5. Verify your recovery phrase by selecting the words in order
  6. Done! Your wallet is ready to use

Your 12-word recovery phrase is the ONLY way to restore your wallet. If you lose both your password and recovery phrase, your funds are LOST FOREVER. We cannot recover them for you.

  • Write your recovery phrase on paper (never digital)
  • Store it in a safe place (safe, vault, secure location)
  • NEVER share it with anyone
  • NEVER take screenshots of it

Sending OCT

  1. Click "Send" button on the main screen
  2. Enter the recipient address (starts with "oct")
  3. Enter the amount to send (up to 6 decimal places)
  4. Select network fee (Slow/Normal/Fast)
  5. Review the transaction details carefully
  6. Click "Send" to confirm

Note: Blockchain transactions are irreversible. Always double-check the recipient address before sending.

Receiving OCT

  1. Click "Receive" button
  2. Your wallet address will be displayed as:
    • Text (you can copy it)
    • QR code (for scanning)
  3. Share your address with the sender
  4. Wait for the transaction to be confirmed (typically 10-60 seconds)

FAQ

Can I recover my wallet if I forget my password?

Yes, IF you have your 12-word recovery phrase. You can import your wallet using the phrase and set a new password.

What if I lose both my password and recovery phrase?

Unfortunately, your funds will be lost forever. There is no password reset or recovery mechanism. This is by design for maximum security.

Is my data sent to any servers?

No. All data stays on your device. The only network requests are to the Octra blockchain RPC node to fetch balance and send transactions.

Can I use this wallet on mobile?

Not yet. Currently only available as a browser extension for desktop. Mobile apps are planned for 2026.

When will the Swap feature be available?

The Swap feature is being developed by the Octra Network team. Expected release: Q1 2026. Check https://docs.octra.org/ for updates.

Can I connect this wallet to DApps?

Not yet. DApp connection (WalletConnect) support is planned for version 2.0.
